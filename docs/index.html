    Atlas Guardrails

    Atlas Guardrails is a CLI tool and MCP server designed to provide deterministic repo context, prevent duplication, and enforce drift guardrails for LLM agents.

    Installation

    npm install
    npm run build
    npm link # Optional, to run 'atlas' globally

    Usage

    CLI Commands

    • Index the repo:

      node dist/index.js index

      Creates .atlas/ directory with symbols.sqlite and manifests.

    • Create a context pack:

      node dist/index.js pack -t "implement authentication" --budget 50000

      Generates pack.json with relevant files and reasoning.

    • Find duplicates:

      node dist/index.js find-duplicates -i "SimpleParser"

      Checks for existing symbols matching the intent.

    • Check for drift:

      node dist/index.js check

      Verifies public API hasn't drifted (if approved_api.json exists).

    • Initialize Ralphy:

      node dist/index.js ralphy-init

      Generates .ralphy/config.yaml.

    • Start MCP Server:

      node dist/index.js mcp

      Starts the MCP server over stdio.

    MCP Configuration

    Add this to your Claude Desktop or Codex configuration:

    {
      "mcpServers": {
        "atlas": {
          "command": "node",
          "args": ["/path/to/atlas/dist/index.js", "mcp"]
        }
      }
    }
